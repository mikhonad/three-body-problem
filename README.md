# Three-Body Problem — Gravitational Animation

Interactive visualization of the classical three-body problem with Newtonian gravity, elastic/inelastic collisions, and orbital mechanics.

**🌐 Live demo:** https://mikhonad.github.io/three-body-problem/

## Features

- Configurable setup: **1–30 stars** and **0–50 planets**
- Realistic gravitational interactions with volume-conserving mergers
- Cinematic auto-camera that follows the action
- Manual zoom & pan (scroll wheel / pinch / drag)
- Unlimited time acceleration
- Mobile-friendly touch controls (pinch-zoom, drag-pan, double-tap reset)
